# Kids' English Coach (Hebrew UI)

MVP v0: show an English word + play audio + 4 Hebrew multiple-choice answers (no nikud).

## Planned features
- Auth + roles: admin/user
- Admin panel: create users
- Per-user progress tracking
- Decks/levels: Level 1 (single-word translations), Level 2 (introduce simple phrases)

## Dev
```bash
npm install
npm run dev
```
